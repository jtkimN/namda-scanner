# namda-scanner

**K-퀀텀 얼라이언스 포럼** 셀프 체크인 시스템.
구글폼 등록 → 뿌리오 알림톡으로 개인 QR 발송 → 행사장 태블릿 스캐너로 입장 확인.

## 구성
| 파일 | 역할 |
|---|---|
| `index.html` | 태블릿 스캐너 (행사장, 2대 동시 지원) |
| `qr.html` | 참가자 개인 QR 페이지 (알림톡 링크가 여는 화면) |
| `generator.html` | QR 생성기 — 인쇄용 명단 카드·재발급 (백업 도구) |
| `apps-script/Code.gs` | 구글 백엔드 (번호·토큰 부여, 알림톡 발송, 체크인 API) |
| `lib/` | jsQR(스캔)·qrcode-generator(생성) — CDN 미사용, 로컬 내장 |

## 견고성 설계 (지난 버전 오류 대응)
- **번호를 "값"으로 조회** → 시트 정렬돼도 엉뚱한 사람 안 뜸
- **LockService** → 스캐너 2대 동시에도 데이터 안전
- **토큰 검증** → 위조·타인 QR 차단
- **JSONP 통신** → 브라우저 CORS/리다이렉트 오류 회피
- **라이브러리 로컬 내장** → CDN 차단/지연에도 스캔 동작
- **오류 원인 화면 표시** → 현장 진단 가능
- QR 형식: `NAMDA1|번호|토큰`

## 검증
- 생성→스캔 라운드트립 16/16 통과 (EC·스케일 전 조합)
- 백엔드 체크인 로직 10/10 통과 (정상·중복·위조토큰·미등록·정렬후 정확성·JSONP)
